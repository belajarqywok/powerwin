# powerwin
tool/command to sleep, restart, and shutdown your PC ( windows )

## how to use
```
 -------------------- powerWIN ---------------------
|                                                   |
|    example : powerwin.exe [OPTIONS.....]          |
|    OPTIONS :                                      |
|    [ -h | --help ] .......... about this tool     |
|    [ -slp | --sleep ] ....... sleep option        |
|    [ -rst | --restart ] ..... restart option      |
|    [ -shd | --shutdown ] .... shutdown option     |
|                                                   |
 ---------------------------------------------------
```
